# solver-sudoku

  Backtracking - Sudoku Solver Implementation

This project is an implementation of a sudoku solver.

## Usage

    python app.py samples/grid2-hard.txt

Just provide a grid as .txt, with in "empty" cells 0 instead.

    000410006
    000800900
    905060003
    007000002
    160000084
    800000300
    700090105
    008002000
    600074000
